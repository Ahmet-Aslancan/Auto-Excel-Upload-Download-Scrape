# Template for Detailed Excel export

**Template.xlsx** in this folder is used automatically by "Download Detailed Excel".

### Important

- **Do not auto-generate / overwrite your template.**
- Put your **real** `Template.xlsx` (the one that already has all dropdowns / data validations) here:
  - `Excel project/templates/Template.xlsx`

The extension loads this exact file, fills only the scraped columns (SN, PEN ID, Name, etc.) into data rows, and leaves **all other columns unchanged** so dropdowns remain available.
